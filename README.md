# Personalized Networking Assistant

The **Personalized Networking Assistant** is an AI-powered web application designed to help users build meaningful connections at professional conferences, networking events, workshops, and social gatherings. By leveraging Natural Language Processing (NLP) and Generative AI, the application creates personalized, context-aware conversation starters based on event descriptions and user interests.

In addition to generating engaging conversation prompts, the application provides **quick fact verification** through Wikipedia APIs, enabling users to confidently discuss relevant topics with accurate, up-to-date information. To encourage continuous improvement, it maintains a **conversation history** and incorporates a **feedback system** that learns from user ratings, helping refine future suggestions.

### Key Features

* 🤖 **AI-Powered Conversation Starters** – Generates personalized, context-aware conversation prompts based on event themes and user interests.
* 🧠 **Theme Extraction** – Uses NLP models to identify key topics from event descriptions for more relevant suggestions.
* ✅ **Quick Fact Verification** – Retrieves concise and reliable information using Wikipedia APIs to support informed conversations.
* 📜 **Conversation History** – Stores previously generated conversation starters for easy access and reference.
* ⭐ **Feedback-Driven Personalization** – Collects user feedback to continuously improve the relevance and quality of future recommendations.
* ⚡ **Fast & Interactive Interface** – Built with **FastAPI** for high-performance backend services and **Streamlit** for an intuitive, responsive user interface.

The project demonstrates the practical application of **Natural Language Processing (NLP)**, **Generative AI**, and **RESTful APIs** to enhance networking experiences by making conversations more engaging, informed, and personalized.
